# BI_2021_Python

**Sys Os modules homework in Bioinformatics Institute**

## Installation
Use python3 to run install.py in terminal:
```
python3 install.py
```

## Usage
Just use utility file like Linux utility:
```
cat.py <file_name>
```
You can use these utilities with pipeline.

## Utilities list
- wc.py with -l -w -c

- ls.py with -a

- sort.py

- rm.py with -r

- grep.py with python regexp

- cat.py

- tail with -n

- mkdir with -p
